###Whiteboard
==========

Whiteboard real-time web app

#### Features

* Real-time
* Multiusers
* Save whiteboard inputs
* Lists of users

#### Requirements

- Node.js v0.10 or higher

```bash
$ brew install node
```

- Coffeescript

```bash
$ sudo npm install -g coffee-script
```


#### Build and run

```bash

$ git clone git@github.com:edanchenkov/whiteboard.git
$ cd whiteboard
$ npm install

$ coffee app.coffee
```
