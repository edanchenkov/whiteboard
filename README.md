###Whiteboard
==========

Whiteboard real-time web app

#### Features

* Real-time
* Multiusers
* Save whiteboard inputs
* Lists of users

#### Requirements

- Node.js v0.10 or higher

``` brew install node```

- Coffeescript

``` sudo npm install -g coffee-script ```


#### Build and run

```bash

$ git clone git@github.com:edanchenkov/whiteboard.git
$ git cd whiteboard
$ npm install

$ coffee app.coffee

```
